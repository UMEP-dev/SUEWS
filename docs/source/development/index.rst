=========================
Development notes
=========================

Archived dashboards and narratives for scientific contributions.

Each entry below corresponds to a merged pull request that introduces or
materially changes a scientific component of SUEWS (physics, numerics,
calibration, benchmarks). The narrative page summarises what changed and
why; the linked dashboard captures the supporting evidence (sensitivity
sweeps, calibration tables, before/after comparisons) at the moment of
merge.

.. note::
   Bulky figures (PNG/JPG) referenced from the dashboards are hosted as
   GitHub Release assets under tags of the form ``dev-notes-gh-<issue>``
   so they remain traceable without bloating the repository.
