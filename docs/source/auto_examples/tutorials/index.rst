Tutorials
---------

Step-by-step tutorials for learning SUEWS urban climate modelling.
Complete these in order, progressing from basics to advanced analysis.

**Learning Path:**

1. **Quick Start** - Run your first SUEWS simulation with sample data
2. **Setup Your Own Site** - Configure parameters for your research location
3. **Impact Studies** - Perform sensitivity analysis and scenario modelling

After completing these tutorials, proceed to the :doc:`/auto_examples/advanced/index`
for model coupling and external integration examples.
