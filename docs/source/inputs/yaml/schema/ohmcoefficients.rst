Ohmcoefficients
===============

**Parameters:**

.. option:: a1 <RefValue[float]>

   OHM coefficient a1: dimensionless coefficient relating storage heat flux to net radiation

   :Unit: dimensionless
   :Default: ``0.0``

.. option:: a2 <RefValue[float]>

   OHM coefficient a2: time coefficient relating storage heat flux to rate of change of net radiation

   :Unit: h
   :Default: ``0.0``

.. option:: a3 <RefValue[float]>

   OHM coefficient a3: constant offset term for storage heat flux

   :Unit: W m^-2
   :Default: ``0.0``

.. option:: ref <Reference (Optional)>

   :Default: Not specified

   For ``ref``, if using the Reference structure, see :doc:`reference` for details.
