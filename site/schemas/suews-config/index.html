Usage in YAML Configuration

# Use specific version:
schema_version: "2025.12"
$schema: "https://umep-dev.github.io/SUEWS/schemas/suews-config/2025.12.json"

# Or use latest (always points to current version):
$schema: "https://umep-dev.github.io/SUEWS/schemas/suews-config/latest.json"

Version Policy

  • Schema versions use CalVer (YYYY.MM) aligned with SUEWS releases
  • Breaking changes result in a new calendar version
  • Older configs are auto-migrated when possible
