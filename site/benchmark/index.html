Each site scores the modelled energy balance against tower observations across SUEWS releases. One site is benchmarked so far — London, across five releases (2025.7 – 2026.4), with no regression detected. Open a site to compare releases. Further sites are planned.

Methodology
  • Each SUEWS release is run on the same site configuration, meteorological forcing and observations, over a fixed evaluation period — only the model version changes, so any difference is attributable to the model.
  • Each release runs a configuration valid in its own schema; performance is the mean absolute error (MAE) and mean bias (MBE) of the modelled energy-balance fluxes (Kup, Lup, QN, QH, QE) against the eddy-covariance observations.
  • A release is compared with the previous one to flag regressions; a site is flagged if error grows materially.
  • Every run is reproducible: re-running a release yields a byte-identical statistics fingerprint.
  • Observations are access-restricted and fetched at run time from a protected archive; only the derived statistics shown here are published.

See the SUEWS documentation for the model description.
