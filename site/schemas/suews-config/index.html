SUEWS Configuration Schema Registry

JSON Schema definitions for SUEWS YAML configuration files. All schema versions are preserved and accessible.

Available Schema Versions (1)

Version 0.1 (Current)

Initial YAML schema with full Pydantic data model (2025.8)

View Schema | Latest | https://suews.io/schemas/suews-config/0.1.json

Usage in YAML Configuration

# Use specific version:
schema_version: "0.1"
$schema: "https://suews.io/schemas/suews-config/0.1.json"

# Or use latest (always points to current version):
$schema: "https://suews.io/schemas/suews-config/latest.json"

# For older versions (if needed for compatibility):
schema_version: "0.1"
$schema: "https://suews.io/schemas/suews-config/0.1.json"

Version Policy

  • Major version (1.0 → 2.0): Breaking changes requiring migration
  • Minor version (1.0 → 1.1): Backward compatible additions
  • Schema versions are independent of SUEWS release versions
