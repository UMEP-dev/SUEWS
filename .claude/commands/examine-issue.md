description: Examine an issue carefully and provide suggestions interactively
---

Invoke the examine-issue skill to analyse an issue thoroughly and provide actionable suggestions. This skill uses an interactive approach for complex issues and direct suggestions for simpler ones. **Never posts anything without explicit approval.**

Usage: `/examine-issue <issue-number>`

Examples:
- `/examine-issue 456` - Examine GitHub issue #456
- `/examine-issue #123` - Also works with # prefix
