SUEWS-related Publications
==========================

This page lists core SUEWS publications from the development team. For
publications from the broader SUEWS user community, see
:ref:`community_publications`.

Browse by topic below, or see the full reverse-chronological list in
:ref:`all_publications`. External pages can deep-link to any topic via
the stable ``#pub-<slug>`` anchors (e.g. ``related_publications.html#pub-energy-balance``).

.. note::

	Papers covering more than one theme appear in every relevant topic
	section, so the topic lists overlap. The :ref:`all_publications`
	section below is de-duplicated. Submission guidance and the
	controlled vocabulary live in the header comment of
	``assets/refs/refs-SUEWS.bib``.
