# Output Variables

SUEWS can output large variable sets, typically grouped by model component.
For quick diagnostics, start with:
- `Kdown` (incoming shortwave radiation);
- `QH` (sensible heat flux);
- `QE` (latent heat flux);
- `QS` (storage heat flux).

Interpret outputs with:
- units and sign conventions;
- averaging windows;
- site context (season, vegetation state, and urban form).

When troubleshooting, compare both aggregate statistics and selected event windows.
