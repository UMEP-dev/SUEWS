# Radiation Methods

SUEWS supports multiple radiation formulations.
The choice affects net radiation and therefore all downstream flux partitioning.

Common options discussed in SUEWS workflows include NARP, BEERS, and SPARTACUS.

Selection considerations:
- data availability for morphology and canopy structure;
- required physical detail versus computational cost;
- consistency with calibration and evaluation datasets.

If changing radiation method, reassess energy-balance closure and diurnal phase behaviour.
